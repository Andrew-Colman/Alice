Alice Synthesis Thirty's Guide
Moderation Commands

For user commands, type "a!help".

<...>: required syntax
[...]: optional syntax

Prefix: "a!"

(Moderator can use Helper commands unless specified otherwise, Owner can use all commands)

1. Helper

- tempmute <user> <duration in seconds, max 1800> <reason>
- verify <user>

2. Moderator

- bulkdelete <amount of messages to delete (2-100)> (only if you have permission in the channel)
- kick <user> [reason]
- mute <user> <reason>
- tempmute <user> <duration in seconds> <reason>

3. Owner

- addtrack <uid>
- ban <user> <reason>
- completepp <user>
- createinvite <expiration time in seconds> <max usage> <reason>
- deletetrack <uid>
- fetchmessage <initial message ID> <message filter>
- ppdelete <user> <which play to delete, in number>
- ppsubmit <user> [a] [b]
  a: number of play to submit (limit at 5 - default at 1)
  b: the position in user's recent play list that you want to start submitting (a + b - 1 < 50 limit)
- ppwipe <user>
- recalc <uid>
- tempban <user> <duration in seconds> <reason>
- unban <user> <reason>
- whitelistremove <map link / map ID>
- whitelistremoveset <set ID>

4. Bot Owner

- changeavatar <number>
- recalcall
- togglestatus <number>
- togglestatus c <activity type> <activity>
